Parcours utilisateur console ARAI
📅 2026-04-09 👤 Aladdin 💻 Desktop 🌐 araipreprod.agence-xr.io
Objectif
Explorer le parcours utilisateur complet de la console ARAI de A a Z et remonter ce qui choque, les incohérences ou manques de clarté.
Charte de test
Inclus : Inscription, Connexion, Présentation, Création et configuration agent, onglets Agents / Bases de connaissances / Conversations / Outils / Documentation, Suppression du compte
Section Action Observation Type
Inscription Champs obligatoires Les champs obligatoires ne sont pas signalés par un astérisque * Friction UX
Inscription Format mot de passe Le format attendu n'est pas précisé avant la soumission (longueur, majuscule, caractères spéciaux...) Friction UX
Presentation Voir la presentation Les diapositives défilent trop rapidement, pas le temps de lire le texte avant de passer a la suivante Friction UX
Creation & Config. agent Choisir une voix Le texte du nom de la voix dépasse du champ de sélection Bug
Creation & Config. agent Ecouter l'apercu des voix Les 3 dernières voix de la liste ne disposent d'aucun aperçu sonore Bug
Creation & Config. agent Comprendre le systeme langue / voix Fonctionnement peu clair : la voix parle-t-elle forcement dans la langue de son drapeau ? La mention des accents entre parenthèses ajoute a la confusion Friction UX
Creation & Config. agent Apercu audio (configuration) L'icône "Ecouter l'aperçu" au-dessus du premier message ne produit aucune action au clic Bug
Creation & Config. agent Support multilingue Certaines langues proposées (ex : Norvégien) n'ont aucune voix correspondante, incohérence entre langues disponibles et voix proposées Bug
Creation & Config. agent Duree maximale de conversation Le libelle Durée maximale d une conversation en secondes (60-3600) manque de clarté, comportement de l'agent en fin de durée non explicite (coupure automatique ?) Friction UX
Creation & Config. agent Tools & Integrations Cette section reste en anglais même lorsque l'interface est paramétrée en français Bug
Creation & Config. agent Creer l'agent (bouton) Apres avoir appuyé sur "Créer Agent", un message pop-up confirme la création mais le bouton reste grisé pendant 8 à 15 secondes avant la redirection vers la page de l'agent Bug
Onglet Agents Trouver le bouton de creation Le bouton "+" de création d'agent est difficile a trouver initialement, un libelle textuel aiderait Friction UX
Onglet Agents Restreindre un agent L'action passe par un clic sur une icone cadenas, du temps a été nécessaire pour le comprendre Friction UX
Onglet Agents Enlever la restriction Complique de retrouver comment retirer la restriction 20 minutes plus tard, le bouton pourrait s'appeler Gérer les restrictions Friction UX
Onglet Agents Onglet Transferer agent Depuis cet onglet on peut voir les agents transférés, mais on ne peut pas en transférer depuis cet endroit Friction UX
Onglet Agents Bouton Modifier vs nom de l agent Cliquer sur le nom de l'agent permet de configurer le partage uniquement. Cliquer sur Modifier donne accès a tout le reste de la configuration, distinction peu évidente Friction UX
Onglet Agents QR code de test Le QR code génère renvoie vers une page d'erreur, impossible de tester l'agent Bug
Onglet Bases de connaissances Restreindre une base de connaissances Comme pour la restriction d'agent, l'action passe par une icone cadenas, du temps a été nécessaire pour le comprendre Friction UX
Onglet Conversations Utiliser le filtre La section Filtrer en haut a droite a un positionnement étrange, elle ne s'intègre pas naturellement dans le contexte de la page Friction UX
Onglet Conversations Pagination Le bouton Suivant reste cliquable a l infini même lorsqu'il n existe pas d'autre page Bug
Onglet Outils Description des outils Les descriptions sont en anglais et ne sont pas entièrement visibles Bug
Onglet Outils Clic sur un outil Les outils sont cliquables, cependant le clic ne produit rien sauf un rafraîchissement de la page Bug
Onglet Documentation Clic sur Documentation La documentation s'ouvre mais l'interface bascule automatiquement en anglais Bug
Onglet Documentation Retour au menu principal Le logo ARAI en haut a gauche de la documentation ne renvoie pas au menu principal, difficile de retrouver son chemin Friction UX
